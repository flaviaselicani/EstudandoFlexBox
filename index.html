Flexbox

Abaixo listei algumas propriedades que mais utilizo. Adicionando propriedades no elemento pai e elementos filhos.
Fontes:
FLEXBOX CSS! Como posicionar elementos na página web
CSS Flexbox Layout Guide

Display: flex

Identificar o elemento container do flexbox (pai). Exemplo(s):
display: flex;
justify-content: space-around;

Display: flex-direction

Direção do eixo principal (como padrão o navegador define para "row"). Exemplo(s):
flex-direction: column;


flex-direction: row;

Justify-Content

Para alinhar os elementos no eixo principal. Exemplo(s):
justify-content: space-between;


justify-content: space-around;

Align-Items

Para alinhar os elementos no eixo perpendicular. Exemplo(s):
display: flex;
flex-direction: column;
align-items: center;

Flex-Wrap

Para definir se os elementos terão uma quebra de linha ao chegarem ao final. Exemplo(s):
display: flex;
flex-wrap: wrap;


display: flex;
flex-wrap: nowrap;
